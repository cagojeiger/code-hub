## Task 시작 체크리스트

다음을 순서대로 확인하세요:

### 1. 현재 위치 확인
- docs/roadmap/ 에서 "Status: In Progress" 파일 찾기
- 해당 파일에서 "Status: In Progress" Milestone 찾기
- 첫 번째 `[ ]` Task가 현재 작업 대상

### 2. 스펙 읽기
- docs/spec.md 에서 관련 섹션 읽기
- docs/architecture.md 에서 컴포넌트 관계 확인

### 3. Exit Criteria 확인
- Task의 `(Exit: ...)` 부분 명확히 이해
- 완료 조건이 측정 가능한지 확인

### 4. 브랜치 생성
- 형식: `feature/{milestone}-{task-name}`
- 예: `feature/m1-docker-setup`
